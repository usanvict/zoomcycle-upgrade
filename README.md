## API list
Created in Bruno

### API for spinning classes

- GET /classes
- GET /classes/:id
- POST /classes
- PUT /classes/:id
- PATCH /classes/:id
- DELETE /classes/:id

### API for users
- GET /users/:id
- PUT /users/:id
- PATCH /users/:id

### API for trainer's classes

- GET /users/:id/classes
- GET /users/:id/classes/:id
- DELETE /user/:id/classes/:id
